# Tic-Tac-Toe

A simple **Tic-Tac-Toe** game built with **HTML, CSS, and JavaScript**. Demonstrates basic DOM manipulation, event handling, and game logic for a two-player game.

## Features

- Two-player gameplay (X vs O)  
- Winner announcement and draw detection  
- Highlights winning combination  
- Reset and New Game buttons  
- Turn indicator
